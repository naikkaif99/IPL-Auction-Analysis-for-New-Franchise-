# 🏏 IPL Auction Analysis for New Franchise

## 📌 Project Overview

This project focuses on analyzing IPL player auction data to help a new franchise build a competitive squad within a given budget. Using data analysis techniques, the project identifies the best player combinations, evaluates player performance, and provides strategic insights for team selection.

---

## 🎯 Objectives

* Analyze IPL auction and player performance data.
* Identify top-performing players across different roles.
* Optimize team selection under budget constraints.
* Build a balanced squad for a new IPL franchise.
* Generate data-driven insights for auction strategies.

---

## 📊 Features

* Player Performance Analysis
* Role-wise Player Evaluation
* Budget Optimization
* Team Composition Insights
* Data Visualization & Reports

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/naikkaif99/IPL-Auction-Analysis-for-New-Franchise-.git
```

2. Navigate to the project folder:

```bash
cd IPL-Auction-Analysis-for-New-Franchise-
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the Jupyter Notebook:

```bash
jupyter notebook
```
